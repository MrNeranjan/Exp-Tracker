# Exp-Tracker

Clean Expo + Expo Router baseline for building an expense tracking app.

## Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the app:

   ```bash
   npm run start
   ```

3. Open target platform from Expo CLI:

   - Android: press `a`
   - iOS: press `i`
   - Web: press `w`

## Current structure

- `app/_layout.tsx`: Root navigation layout
- `app/index.tsx`: Initial home screen

This repo has been stripped of the default template demo files to keep implementation focused and clean.
